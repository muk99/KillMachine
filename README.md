パラメータの解説
----------------
入力信号を指定された時間でオン・オフをひたすら繰り返すだけのエフェクターです。
+   `Dead Length` :
    音を切る(スイッチオフにする)にする時間の長さ
 
+   `Alive Length` :
    音を鳴らす(スイッチオンにする)にする時間の長さ
